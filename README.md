# 我的ArchLinux安装脚本

### **官方推荐的安装方式依然是手动安装**



#### 一. 安装步骤

在安装之前，你需要在live环境安装git

```shell
pacman -Syu git
git clone https://hub.fastgit.xyz/AliceQAQ/Kuku_archlinux_installScripts
cd Kuku_archlinux_installScritps
# 根据你的需求更改好脚本相关内容并保存
bash archinstall.sh
```



#### 二. 安装图形界面

```shell
git clone https://hub.fastgit.xyz/AliceQAQ/Kuku_archlinux_installScripts
cd Kuku_archlinux_installScritps
# 根据你的需求更改好脚本相关内容并保存
bash arch_graphics.sh
```
